Modular CSS for Hugo

24.04.2021 21:46 modular css bem

Godular is a Hugo starter site using Modular CSS. 1. Frontend build Modular CSS is a way of organising your CSS, and other assets, into discrete and re‑usable chunks. The individual chunks are concatenated, transformed and optimised. Godular uses Hugo Pipes with PostCSS. 2. Code organisation The code organisation is based on Inverted Triangle CSS by Harry Roberts, from the most applicable styles (variables, base) to specific modules (objects and components) and to very specific trumping utilities and shameful hacks.
